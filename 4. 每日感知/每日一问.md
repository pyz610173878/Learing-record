# 每日一问

今天分析的是常见的各大视频网站上面的**广告**的显示。



# 不同之处

1. 不能拖动进度条。

2. 广告类似与一个超链接。

    1.1 可以点击，并且跳转。

    1.2 大多数情况下，六十秒内由多个小视频组成，随着视频变换，跳转的html页面也会随之改变。

    1.3 没有调解倍速的功能。

3. 右上角有一个倒计时功能，和开通会员的a标签。

    



# 相同之处

1. 可以全屏放大。
2. 音量的功能只有 两个状态 开与关，这个类似于有限状态机[FSM]不能调解音量大小。





# 可挖掘的技术点

* 广告时间的计数器是怎么弄的
* 如何做到html页面跳转与视频绑定。
