基本上，您将学习到如何选取 HTML 元素，以及如何对它们执行类似隐藏、移动以及操作其内容等任务

jQuery 是一个 JavaScript 函数库。

jQuery 库包含以下特性：

-   HTML 元素选取
-   HTML 元素操作
-   CSS 操作
-   HTML 事件函数
-   JavaScript 特效和动画
-   HTML DOM 遍历和修改
-   AJAX
-   Utilities

总结:

`jquery`是一个JavaScript函数库。可以联想JavaScript中存在的标准库。只是提供的功能不同。

此函数库的特性:

1.  能够获取，修改，删除，访问html中的元素，即元素的样式。
2.  有更简单的方法绑定，添加事件。(提供事件函数)
3.  提供了css动画。
